BiteBase 🍔
BiteBase is a web-based food ordering management system designed to streamline the process of managing customer orders for a restaurant or food stall.

Features
User Menu: Customers can browse available dishes and add them to their cart.

Shopping Cart: Allows users to review items, adjust their selection, and finalize their orders.

Order Tracking: New orders are automatically set to "Pending" status.

Management Portal: Admins can view all orders and update their status from "Pending" to "Delivered."

Technologies Used
Frontend: HTML, CSS

Backend: PHP

Database: MySQL (managed via phpMyAdmin)

Server: XAMPP (Apache & MySQL)

Setup Instructions
1. Prerequisites
Ensure you have XAMPP installed and running on your local machine.

2. Installation
Clone or Download this repository into your htdocs folder:

Path: C:\xampp\htdocs\bitebase

Start Services: Open the XAMPP Control Panel and start Apache and MySQL.

Setup Database:

Open your browser and navigate to http://localhost/phpmyadmin/.

Create a new database named bitebase.

Click the Import tab at the top.

Choose the bitebase.sql file included in this repository and click Go.

Access the Application:

Open your browser and go to http://localhost/bitebase/index.php.
